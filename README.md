## ❓ Troubleshooting

**Mac: "cannot be opened because it cannot be verified"**
→ Right-click the file → Open → Click "Open" in the dialog

**Mac: "permission denied"**
→ Open Terminal → type `chmod +x ` → drag the .command file in → press Enter → double-click again

**Windows: winget not found**
→ Update Windows to version 10/11 → try again, or install Python and FFmpeg manually (links in the installer)

**Processing seems stuck**
→ Large videos (1GB+) take 15–30 minutes. The progress bar shows real-time %. Leave it running.
